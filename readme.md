https://exvault.github.io/DigitalBoardGameList

List of digital board game adaptations.

The project tracks digital releases across platforms and publishes available information as a static table. Data is collected through public APIs or scraping, and includes BoardGameGeek metadata, store links, prices, discounts, DLC information, and update details where available.

The table data is updated every 24 hours.

Prices are currently fetched for:

- Google Play
- App Store
- Steam

Discounts are currently detected for:

- Google Play
- Steam

The game catalog is maintained in `games.yaml`.
